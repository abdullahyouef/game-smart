# لعبة الذاكرة - مطابقة البطاقات

لعبة ذاكرة تفاعلية حيث يحاول اللاعب العثور على الأزواج المتطابقة من البطاقات.

## كيفية اللعب

1. انقر على زر "بدء اللعبة" لبدء المؤقت
2. انقر على البطاقات لقلبها وكشف الرموز تحتها
3. ابحث عن زوجين متطابقين من الرموز
4. حاول إكمال اللعبة بأقل عدد من الحركات وأسرع وقت ممكن
5. استخدم زر "إعادة" لإعادة تشغيل اللعبة

## المميزات

- عداد الوقت لقياس سرعة أدائك
- عداد الحركات لتتبع عدد المحاولات
- واجهة مستخدم بسيطة وسهلة الاستخدام
- تصميم متجاوب يعمل على جميع الأجهزة
- تأثيرات بصرية عند قلب البطاقات وعند التطابق

## التقنيات المستخدمة

- HTML5
- CSS3 (بما في ذلك Grid وAnimations)
- JavaScript (ES6+)

## المتطلبات

متصفح ويب حديث يدعم JavaScript وCSS Grid
